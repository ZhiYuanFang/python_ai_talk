## ADDED Requirements

### Requirement: 历史查询接口
系统 SHALL 在意图分析接口中支持历史查询场景，当识别到历史查询意图时，检索并返回相关历史记录。

#### Scenario: 查询特定类型记录
- **WHEN** 用户输入"今天喝了多少奶粉"，意图分析结果为 search
- **THEN** 系统检索今天的奶粉喂养记录并返回

#### Scenario: 查询所有记录
- **WHEN** 用户输入"查看所有记录"，意图分析结果为 search
- **THEN** 系统检索所有类型的历史记录并返回

#### Scenario: 查询最近记录
- **WHEN** 用户输入"上次喂奶是什么时候"，意图分析结果为 search
- **THEN** 系统检索最近一条喂养记录并返回

### Requirement: 历史记录格式化
系统 SHALL 将历史记录格式化为自然语言文本返回给用户。

#### Scenario: 格式化单条记录
- **WHEN** 系统检索到单条历史记录
- **THEN** 系统将记录格式化为自然语言文本（如"上次喂奶是今天上午10点，喂了120ml奶粉"）

#### Scenario: 格式化多条记录
- **WHEN** 系统检索到多条历史记录
- **THEN** 系统将记录分组汇总，格式化为自然语言文本

### Requirement: LLM 总结历史
系统 SHALL 使用 LLM 对历史记录进行总结，生成自然语言回答。

#### Scenario: 总结今日喂养
- **WHEN** 用户询问"今天吃了多少"
- **THEN** 系统检索今天的喂养记录，使用 LLM 生成总结回答

#### Scenario: 总结一周喂养
- **WHEN** 用户询问"最近一周的喂养情况"
- **THEN** 系统检索最近一周的喂养记录，使用 LLM 生成总结回答
